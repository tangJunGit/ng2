### npm scripts

```bash
# 安装
npm install

# 启动服务
# url: http://localhost:3000
npm start

# 通过 Jit 编译项目
npm run build

#通过 Aot 编译项目
npm run ngc
npm run build:aot

# 通过 rollup 编译项目
npm run rollup    

# 打包后启动服务
# url: http://localhost:8080
npm run server

# 测试
npm test
